# Proyecto appcomercio
Sistema local de comercio desarrollado en WordPress bajo XAMPP.
Incluye tema personalizado y plugin de seguimiento documentario.

## Requisitos
- XAMPP 8.x
- WordPress 6.x
- PHP 8.2
- MySQL 8.0
